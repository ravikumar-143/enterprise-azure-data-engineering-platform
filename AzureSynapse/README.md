
# Azure Synapse Analytics

## Overview

Azure Synapse Analytics provides the serving layer for the project by exposing curated datasets stored in Azure Data Lake Storage Gen2.

## Implemented Components

- Database Scoped Credential
- External Data Source
- External File Format
- SQL Views
- OPENROWSET
- External Tables

## Output

Business-ready datasets are exposed through Serverless SQL Pool for reporting and analytics.

## Power BI Integration

Power BI connects to the Azure Synapse Serverless SQL Endpoint to visualize curated datasets.
